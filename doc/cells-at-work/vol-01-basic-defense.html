01

肺炎球菌肺炎球菌 / Pneumococcus

微生物学自然免疫呼吸系统

观察域：微生物学 / 自然免疫 / 呼吸系统

作品观察

赤血球在肺部迷路，肺炎球菌趁机侵入并试图经血流扩散，白血球（好中球）追踪并吞噬病原。

拟人化解释

肺炎球菌被画成会伪装、会潜入运输网络的入侵者；好中球的“巡逻—追踪—吞噬”是自然免疫第一线的戏剧化。

现实机制

肺炎链球菌常定植鼻咽，机会性进入肺泡或血流后可导致肺炎、菌血症、脑膜炎；荚膜帮助其逃避免疫识别，好中球通过趋化、吞噬和杀菌颗粒清除细菌。

易误解点 / 安全提醒

现实中肺炎的严重程度与年龄、免疫状态、基础病有关；抗生素和疫苗是现代医学的重要工具，漫画中的“单挑”只是免疫反应的拟人化。

◎ 迁移记忆把“肺泡—毛细血管—血流”想成交通枢纽，可理解为什么呼吸道病原能从局部感染发展为全身感染。

Sample CH-02 屏障/炎症
02

杉花粉过敏スギ花粉アレルギー / Cedar Pollen Allergy

过敏获得免疫炎症介质

观察域：过敏 / 获得免疫 / 炎症介质

作品观察

无害的杉花粉被免疫系统误判成强敌，B细胞、肥大细胞和组织胺造成一场“过度防御”。

拟人化解释

肥大细胞像严格按说明书开闸的管理员，组织胺被表现为大规模城市灾害。

现实机制

I型超敏反应中，IgE结合肥大细胞表面受体；再次接触过敏原后交联触发脱颗粒，释放组胺、白三烯等介质，引起喷嚏、鼻水、瘙痒和黏膜水肿。

易误解点 / 安全提醒

过敏不是“免疫太弱”，而是识别和反应阈值失衡；抗组胺药、局部糖皮质激素、免疫治疗等需按医嘱使用。

◎ 迁移记忆用“误报的火警系统”理解过敏：警报是真的，目标却未必危险。

Sample CH-03 病原/感染
03

流感インフルエンザ / Influenza

病毒学细胞免疫疫苗

观察域：病毒学 / 细胞免疫 / 疫苗

作品观察

流感病毒感染体细胞后迅速复制；初出任务的T细胞在树突状细胞鼓励下成长，参与清除感染细胞。

拟人化解释

病毒感染细胞被画成“同化/僵尸化”，T细胞训练与战斗对应克隆扩增和效应分化。

现实机制

流感病毒侵入上皮细胞并复制，先由干扰素、NK细胞等自然免疫压制，随后抗原呈递激活T细胞和B细胞；抗原漂移使季节性疫苗需更新。

易误解点 / 安全提醒

流感与普通感冒不同，老人、孕妇、慢病患者风险更高；抗生素不能治疗病毒感染，抗病毒药有时间窗。

◎ 迁移记忆把“被病毒占据的细胞”理解为需要切断的复制工厂，才能理解为什么细胞毒性T细胞会杀伤自体感染细胞。

Sample CH-04 屏障/炎症
04

擦伤すり傷 / Scrape Wound

皮肤屏障止血炎症

观察域：皮肤屏障 / 止血 / 炎症

作品观察

表皮破损打开了体内世界的城门，血小板搭建凝血栓，白血球与细菌在伤口前线交战。

拟人化解释

血小板被画成施工队，纤维蛋白网像脚手架，贴切表现止血与修复。

现实机制

皮肤屏障破裂后，血管收缩、血小板黏附聚集、凝血级联生成纤维蛋白；随后炎症细胞清除病原和坏死组织，成纤维细胞与上皮细胞参与修复。

易误解点 / 安全提醒

伤口处理不是“越刺激越好”：清洁、止血、覆盖和观察感染迹象更关键；深伤、动物咬伤或污染伤需专业评估。

◎ 迁移记忆把止血分为“临时堵漏（血小板）—加固水泥（纤维蛋白）—清场修路（炎症与修复）”。
