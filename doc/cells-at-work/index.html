推荐阅读路径

  1. 只想补免疫基础：第1卷 → 第3卷 → 第5卷第21–25话。
  2. 想看临床与公共卫生：第4卷失血休克 → 第6卷左移/iPS/新冠 → 特别宣传篇。
  3. 想扩展身体语境：BLACK（成人不健康）/ LADY（女性身体）/ BABY（新生儿）/ 药（药理）。
  4. 想做课堂展示：使用本页ECharts领域图、各卷Mermaid流程图、术语表和参考页。

医学阅读边界

《工作细胞》的强项是用拟人化降低理解门槛；弱点是为了叙事会压缩时间、简化细胞数量、突出“战斗”。本手册会尽量标出易误解点：不要把漫画情节当诊疗建议，不要自行用药，不要将“免疫力越强越好”作为健康目标。
