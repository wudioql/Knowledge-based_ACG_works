幽门螺杆菌ピロリ菌 / H. pylori

胃黏膜细菌适应消化系统

观察域：胃黏膜 / 细菌适应 / 消化系统

作品观察

幽门螺杆菌在胃酸环境中生存并攻击胃黏膜，乳酸菌等角色引出胃肠微生态。

拟人化解释

能在强酸“荒漠”中生存的敌人，表现幽门螺杆菌的尿素酶和黏液层适应。

现实机制

H. pylori通过尿素酶产生氨中和局部酸性，定植胃黏膜，和慢性胃炎、消化性溃疡及部分胃癌风险相关；根除治疗通常需联合用药。

易误解点 / 安全提醒

是否检测和根除应结合症状、溃疡史、地区指南与医生建议；益生菌不能替代根除治疗。

◎ 迁移记忆理解“胃酸强”不等于“胃内无菌”。

Sample CH-21 病原/感染
21

抗原变异抗原変異 / Antigenic Shift

流感变异免疫逃逸疫苗

观察域：流感变异 / 免疫逃逸 / 疫苗

作品观察

既有记忆对新变体不完全有效，免疫系统面对“换装”的流感病毒需要重新组织。

拟人化解释

病毒换外套对应表面抗原改变；记忆细胞的档案可能失灵。

现实机制

流感A病毒可发生抗原漂移（点突变积累）和抗原转换/重配（不同病毒株基因片段重组），后者可能带来大流行潜力。

易误解点 / 安全提醒

变异并不必然更毒，但会影响传播和免疫保护；疫苗监测需全球协作。

◎ 迁移记忆区分“微调版升级（漂移）”与“模块重组（转换）”。

细胞因子サイトカイン / Cytokines

免疫通讯炎症放大信号网络

观察域：免疫通讯 / 炎症放大 / 信号网络

作品观察

细胞因子像广播/邮件在免疫细胞间传递指令，也可能引发过度动员。

拟人化解释

树突状细胞和辅助T细胞的信息传递强调免疫不是单兵战，而是网络。

现实机制

细胞因子包括白介素、干扰素、肿瘤坏死因子、趋化因子等，调控免疫细胞募集、活化、增殖和分化；过度释放可造成系统性炎症。

易误解点 / 安全提醒

“提高免疫力”不是越强越好，失控的炎症同样伤身。

◎ 迁移记忆把免疫系统看成有多频道通讯的应急指挥中心。

23

有害菌/恶玉菌悪玉菌 / Harmful Bacteria

肠道菌群生态平衡短链脂肪酸

观察域：肠道菌群 / 生态平衡 / 短链脂肪酸

作品观察

肠道内善玉菌、恶玉菌和机会菌争夺生态位，饮食和环境改变左右局势。

拟人化解释

细菌阵营的领地战让“菌群平衡”变得可视化。

现实机制

肠道菌群参与膳食纤维发酵、短链脂肪酸生成、屏障维护和免疫调节；所谓善/恶/日和见并非固定道德标签，而取决于菌种、位置和宿主状态。

易误解点 / 安全提醒

益生菌产品效果具有菌株和人群差异；均衡饮食、纤维摄入和合理用药是基础。

◎ 迁移记忆把菌群理解为生态系统，不是黑白名单。

Sample CH-24 肿瘤免疫
24

癌细胞2（前篇）がん細胞２（前編） / Cancer Cell II Part I

肿瘤复发免疫逃逸微环境

观察域：肿瘤复发 / 免疫逃逸 / 微环境

作品观察

癌细胞以更复杂方式回归，并利用肠道失衡和免疫调节机制增强自己。

拟人化解释

制御性T细胞被利用、恶玉菌环境助推癌细胞，是肿瘤微环境的形象化。

现实机制

肿瘤复发与残存细胞、免疫选择、抗原改变和微环境有关；Treg、免疫检查点、炎症和代谢环境都可能影响抗肿瘤免疫。

易误解点 / 安全提醒

漫画强调免疫战，但现实肿瘤治疗需要筛查、病理分型、手术/放疗/药物/免疫治疗综合判断。

◎ 迁移记忆癌症不是单个坏细胞，而是一套生态位和逃逸策略。

25

癌细胞2（后篇）がん細胞２（後編） / Cancer Cell II Part II

记忆T细胞菌群免疫肿瘤控制

观察域：记忆T细胞 / 菌群免疫 / 肿瘤控制

作品观察

乳酸菌、普通细胞和免疫细胞的协作改变局面，癌细胞再次被压制。

拟人化解释

“善玉菌支援免疫”把菌群—免疫—肿瘤的复杂关联转为清晰故事线。

现实机制

研究显示肠道菌群可影响免疫治疗反应和炎症状态，但机制复杂；记忆T细胞能在再次遭遇抗原时更快反应。

易误解点 / 安全提醒

不能把“吃益生菌”理解为防癌或治癌方案；生活方式只是风险管理的一部分。

◎ 迁移记忆从“局部微生态影响全身免疫阈值”理解肠道为何常被称作免疫器官。
