全8卷2018–2021
成人健康风险医学干预生活方式

工作细胞BLACK / はたらく細胞BLACK / CODE BLACK

作者/制作：原田重光/初嘉屋一生，清水茜监修

知识视角：吸烟、饮酒、压力、性传播感染、糖尿病、心梗、抑郁、癌症治疗等成人健康风险。

卷主题地图
1吸烟、肝脏酒精、性功能/淋病、过劳脱发
2脚气、胃溃疡、痛风、心绞痛/心肌梗死
3输血到第二身体、肾结石、尿路感染、肺栓塞
4糖尿病、糖化、痔、带状疱疹
5坏疽、截肢、睡眠呼吸暂停、牙周病、药物/酒精
6洗胃、脑萎缩、抑郁、戒烟与药效
7癌症、转移、手术、放疗、抗癌药、免疫治疗
8复兴、凝血、造血分化、脑转移与终局

学习抓手

  • 生活方式因素会转化为细胞层面的慢性压力。
  • 药物和手术频繁登场，适合讨论疗效与副作用。
  • 不适合低龄读者单独阅读，可作为成人健康教育材料。

急性风险毒性/依赖社会风险

工作细胞ILLEGAL / はたらく細胞イリーガル

作者/制作：桥本カヱ/次恒一，清水茜监修

知识视角：以违法药物、暴力伤害、极端环境为切口，展示“身体被外部行为裹挟”时细胞如何应对。

卷主题地图
1药物乱用与急性应激
2伤害、感染风险与血液循环
3依赖、戒断与精神/神经影响
4收束：风险行为的身体代价

学习抓手

  • 毒性、剂量、依赖和社会风险要同时讨论。
  • 可与BLACK比较：一个是生活方式慢性黑化，一个是非法/暴力急性化。
全4卷2023–2025
药理疫苗抗菌/耐药

工作细胞 药 / はたらく細胞 おくすり / Medicine

作者/制作：割田コマ/九似良，清水茜监修

知识视角：药物拟人化：镇痛退热、抗菌、抗炎、抗组胺、疫苗等如何在体内发挥作用。

卷主题地图
1药物登场：吸收、分布、作用点
2镇痛退热与常见症状
3感染、抗菌药与耐药意识
4抗菌/抗炎、抗组胺、五种混合疫苗，完结

学习抓手

  • 药物不是魔法，而是有靶点、剂量、时间窗和副作用。
  • 疫苗是预防性训练，不是感染后万能治疗。
