15

金黄色葡萄球菌黄色ブドウ球菌 / Staphylococcus aureus

机会致病菌皮肤感染毒素

观察域：机会致病菌 / 皮肤感染 / 毒素

作品观察

强大的金黄色葡萄球菌带领细菌群突破屏障，免疫细胞需要协同作战。

拟人化解释

葡萄球菌女王化的形象表现其毒素、黏附和免疫逃逸能力。

现实机制

金黄色葡萄球菌可定植皮肤鼻腔，也可引发脓疱、蜂窝织炎、肺炎、败血症；凝固酶、毒素和生物膜帮助致病，耐甲氧西林菌株是临床挑战。

易误解点 / 安全提醒

抗菌药选择需考虑部位和耐药性；反复皮肤脓肿要评估卫生、接触传播和基础病。

◎ 迁移记忆记住“常在菌≠永远无害”，环境改变即可转为机会致病。

Sample CH-16 病原/感染
16

登革热デング熱 / Dengue Fever

虫媒病毒血小板公共卫生

观察域：虫媒病毒 / 血小板 / 公共卫生

作品观察

蚊媒病毒入侵后引发发热与血小板危机，身体防线陷入高强度应答。

拟人化解释

蚊媒传播把“外界生态”接入体内战场，提醒疾病并非只发生在身体内部。

现实机制

登革病毒由伊蚊传播，可导致发热、皮疹、肌痛、白细胞和血小板下降；再次感染不同血清型时存在严重化风险机制（抗体依赖增强）的讨论。

易误解点 / 安全提醒

登革热没有特效抗病毒药，重在防蚊、补液和监测出血/休克警讯；避免自行使用增加出血风险的药物。

◎ 迁移记忆把传染病链条拆成病原体、媒介、宿主、环境四环。

Sample CH-17 血液/循环
17

失血性休克（前篇）出血性ショック（前編） / Hypovolemic Shock I

出血休克红细胞

观察域：出血 / 休克 / 红细胞

作品观察

大量出血使运输网络崩溃，赤血球在低灌注环境中仍试图送氧。

拟人化解释

道路坍塌和居民缺氧对应血容量下降、组织灌注不足。

现实机制

急性失血减少循环血量和静脉回流，心输出量下降，交感神经代偿导致心率增快、外周血管收缩；组织缺氧可走向乳酸酸中毒和器官损伤。

易误解点 / 安全提醒

失血性休克需立即止血、补液、输血和病因处理；单靠“细胞努力”无法逆转。

◎ 迁移记忆休克不是“情绪惊吓”，而是组织灌注不足的综合征。

Sample CH-18 血液/循环
18

失血性休克（后篇）出血性ショック（後編） / Hypovolemic Shock II

输血凝血急救

观察域：输血 / 凝血 / 急救

作品观察

外部输血带来新的红细胞，血小板和凝血系统继续封堵破口，体内获得恢复机会。

拟人化解释

外来红细胞的到来让读者理解输血是把运氧能力和循环容量重新带回系统。

现实机制

创伤出血处理包括控制出血源、维持气道呼吸循环、成分输血、纠正凝血障碍和低体温。输血需匹配血型并监测反应。

易误解点 / 安全提醒

现实急救中的黄金时间和专业系统非常关键；不要把补水等同于治疗严重失血。

◎ 迁移记忆用“容量、载氧、凝血、温度、钙离子”五要素理解大出血复苏。

Sample CH-19 免疫学
19

派尔集合淋巴结パイエル板 / Peyer's Patch

肠道免疫黏膜屏障IgA

观察域：肠道免疫 / 黏膜屏障 / IgA

作品观察

免疫细胞在肠道淋巴组织侦察抗原，守住食物与微生物交界面。

拟人化解释

派尔板像情报关口，M细胞/树突状细胞把腔内信息递交免疫系统。

现实机制

派尔集合淋巴结属于肠相关淋巴组织，参与抗原采样、B细胞活化和IgA应答，在耐受食物/共生菌与防御病原间维持平衡。

易误解点 / 安全提醒

肠道免疫的目标不是“杀光细菌”，而是辨别、共处和必要时清除。

◎ 迁移记忆黏膜免疫是“边境管理”：吞吐量大，所以更依赖分级筛查。
