Sample CH-10 血液/循环
10

血液循环血液循環 / The Circulatory System

循环系统气体运输红细胞

观察域：循环系统 / 气体运输 / 红细胞

作品观察

赤血球完成从肺到全身再返回肺的送货路线，读者跟随她认识体循环和肺循环。

拟人化解释

氧气和二氧化碳被拟作包裹，血管网络像城市道路。

现实机制

右心将静脉血送往肺进行气体交换，左心将含氧血泵向全身；红细胞血红蛋白可结合氧并运输部分二氧化碳。

易误解点 / 安全提醒

现实血液循环是海量细胞同时运行，不会像漫画角色那样频繁相遇；但路线图非常适合作为入门记忆。

◎ 迁移记忆用“右心—肺—左心—全身—右心”的闭环背诵循环。

Sample CH-11 病原/感染
11

普通感冒综合征風邪症候群 / The Common Cold

呼吸道病毒黏膜免疫症状机制

观察域：呼吸道病毒 / 黏膜免疫 / 症状机制

作品观察

普通细胞因无聊而被病毒利用，体内以咳嗽、喷嚏和免疫清除来结束骚动。

拟人化解释

被病毒操控的一般细胞体现病毒必须借宿主细胞复制。

现实机制

普通感冒多由鼻病毒等引起，上呼吸道黏膜感染后释放炎症介质，引发鼻塞、喷嚏、咽痛；症状很大一部分来自免疫反应而非组织破坏。

易误解点 / 安全提醒

感冒多数自限，重点是休息、补液和对症；高热持续、呼吸困难、基础病患者应警惕并发症。

◎ 迁移记忆区分“病原复制”和“身体清除病原时制造的症状”。

Sample CH-12 免疫学
12

胸腺细胞胸腺細胞 / Thymocytes

T细胞发育免疫耐受选择

观察域：T细胞发育 / 免疫耐受 / 选择

作品观察

杀手T细胞与辅助T细胞的学生时代揭示胸腺内严苛筛选。

拟人化解释

胸腺学校的考试对应正选择与负选择：既要会认MHC，又不能强烈攻击自我。

现实机制

T细胞前体在胸腺中重排TCR，经正选择保留能识别自身MHC者，经负选择删除强烈自反应者，建立中枢耐受。

易误解点 / 安全提醒

自身免疫疾病不能简单归因于“训练失败”，遗传、感染、环境和外周调节共同参与。

◎ 迁移记忆“会识别身份证，但不能攻击自己人”是理解T细胞选择的核心。

Sample CH-13 免疫学
13

获得性免疫獲得免疫 / Acquired Immunity

抗体记忆细胞免疫二次应答

观察域：抗体 / 记忆细胞 / 免疫二次应答

作品观察

记忆细胞回想旧敌，B细胞生产抗体，显示再次遭遇病原时的快速反应。

拟人化解释

记忆细胞的夸张预言和B细胞的抗体炮，分别对应免疫记忆和体液免疫。

现实机制

抗原呈递与辅助T细胞信号帮助B细胞活化、克隆扩增、亲和力成熟并分化为浆细胞和记忆B细胞；再次感染时反应更快更强。

易误解点 / 安全提醒

免疫记忆有时会随时间下降，病原变异也会降低保护；这就是加强针和疫苗更新的原因之一。

◎ 迁移记忆把疫苗理解为“安全演习”：先建档案，再面对真敌。

Sample CH-14 微生态
14

痤疮/青春痘ニキビ / Acne

皮肤微生态炎症皮脂腺

观察域：皮肤微生态 / 炎症 / 皮脂腺

作品观察

毛孔内皮脂和痤疮相关菌群引发局部炎症，免疫细胞介入清场。

拟人化解释

毛孔像拥堵地下街，皮脂堆积和细菌增殖让局部环境失衡。

现实机制

痤疮与皮脂分泌、毛囊角化、Cutibacterium acnes、炎症反应相关；并非单纯“脏”或“上火”。

易误解点 / 安全提醒

挤压会加重炎症和瘢痕风险；中重度痤疮可用外用维A酸、过氧化苯甲酰、抗菌药或口服治疗。

◎ 迁移记忆从“油脂—角栓—菌群—炎症”四步理解痤疮。
